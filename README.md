# LeMay Research

Technical research and architecture documentation from LeMay Inc.

## Getting Started

### Prerequisites

- Node.js 18.0 or higher
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

### Production Build

```bash
npm run build
npm run start
```

## Structure

- `pages/` - MDX documentation pages
- `public/` - Static assets
- `theme.config.js` - Nextra theme configuration
- `next.config.mjs` - Next.js configuration
